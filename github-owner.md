# 序

リポジトリは private にする。
メンバー (collaborator) は 
（fork ではなく） clone をつかって
共同開発に参加する。
  
このページはリポジトリ作成者（オーナー）がすべきことを
書いている。
メンバーのやるべきことは
[[github-member]] を見よ。

あなたのアカウントの名前を foo とします。
メンバーの名前を bar とします。
対象となるリポジトリを baz とします。
