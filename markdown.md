# はじめに

MSWord は捨てましょう。

Markdown を使いましょう。
とっても簡単です。

# 5分間入門

お好きなエディタをつかってください。
`bar.md` （名前はなんでもいいです）[^fname]
というファイルを新規作成します。

[^fname]: 日本語のファイル名は使わないようにしましょう。

まずはタイトルを書きましょう。

```markdown
---
title: 人類学を自然化する
subtitle: ミリカンの誤解
author: Satoshi Nakagawa
---

```

つぎに、だいたいの枠組をつくってしまいましょう。

```markdown

# 序

# ミリカンの哲学

## ミリカンはだれか

## ミリカンの哲学

# ミリカンと人類学

## ANTという人類学

## どうしたこうした

# 結論

# 参考文献


```
